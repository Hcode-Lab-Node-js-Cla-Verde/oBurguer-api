## Description

This project was developed by the Green Clan students of the course [Hcode Lab Backend Node.js developer](https://www.hcodelab.com.br/).

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
